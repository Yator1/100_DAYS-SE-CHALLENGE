Introduction

Python is an interpreted, high-level and general-purpose programming language. Python's design philosophy emphasizes code readability with its notable use of significant whitespace.

Its language constructs and object-oriented approach aim to help programmers write clear, logical code for small and large-scale projects.

print("Hello, Python!")
  • Created by Guido van Rossum
  • First released in 1991
  • Open-source software

Python is dynamically typed and garbage-collected. It supports multiple programming paradigms, including structured, object-oriented, and functional programming.

Getting Started

To get started with Python, you need to install it on your machine. Python installers are available for various operating systems.

Download the latest version from the official Python website: python.org

python --version
  • Download Python
  • Install Python
  • Verify the installation

Once installed, you can start writing Python code using the command line or an Integrated Development Environment (IDE) like PyCharm or VS Code.

Data Types

Python has several built-in data types that are used to store data of different kinds.

Common data types include integers, floating-point numbers, strings, lists, tuples, sets, and dictionaries.

my_string = "Hello, World!"
  • int: Integer
  • float: Floating-point number
  • str: String
  • list: List
  • dict: Dictionary

Each data type has specific methods and operations associated with it, allowing for efficient data manipulation.

Control Flow

Control flow statements allow you to execute code based on certain conditions and to repeat code using loops.

Common control flow statements include if-else, for loops, and while loops.

if x > 0: print("Positive") else: print("Negative or Zero")
  • if-else: Conditional execution
  • for: Iteration over a sequence
  • while: Repeated execution as long as a condition is true

Using control flow statements effectively is crucial for implementing logic in your programs.

Functions

Functions are reusable blocks of code that perform a specific task. They help in making your code modular and organized.

In Python, functions are defined using the def keyword.

def greet(name): return f"Hello, {name}!"
  • Define a function with def
  • Call a function by its name
  • Return values using return

Functions can take arguments and return values, making them highly versatile for various programming tasks.
